# Egg-Buzzer
Egg-Buzzer is a Littlebird Electronics Egg module with a buzzer attached, this is the python library to make it work with a Raspberry Pi.

# Usage

```
import time
import Egg-Buzzer
buzzer = new Buzzer(pin)
buzzer.Frequency(freq in Hz)
time.sleep(2)
buzzer.Stop()
```
